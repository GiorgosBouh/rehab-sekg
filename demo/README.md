# Static SEKG demo

This folder contains a static demo page for the rehabilitation SEKG
proof-of-concept.

The page includes tabs for representative records, the compact graph
view, the enterprise model, SPARQL evaluation results, and ontology
artifacts. The Enterprise tab shows departments, roles, responsibilities,
the care pathway, governance flow, standards positioning, and the value
proposition represented in the graph.

The Ontology tab also links to reviewer-facing documentation in `docs/`,
including enterprise model views, the LLM modeling protocol, and
standards alignment.

Open `index.html` directly in a browser, or publish the repository with
GitHub Pages and point it to this folder.

The demo has no build step, no backend, and no external runtime
dependencies. It uses relative links to the ontology, instance, sample,
and evaluation artifacts in the repository.
